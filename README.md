An all-in-one travel assistant Android app to make your trips smarter, smoother, and stress-free. Plan itineraries, track expenses, get live weather, convert currency, navigate destinations, and more – all from one app.

---

## ✨ Features

- 📅 **Trip Planner** – Add and manage your travel itineraries.
- 💸 **Expense Tracker** – Track and manage expenses using Firebase Firestore.
- 🌦️ **Weather Updates** – Get current weather and a 7-day forecast using Open-Meteo API.
- 🔄 **Currency Converter** – Live currency rates powered by ExchangeRate-API.
- 🗺️ **In-App Navigation** – Navigate directly in-app using Google Maps SDK & Directions API.
- ✅ **To-Do List** – Organize your travel goals and essentials.
- 📦 **Packing List** – Save and manage what to carry using SharedPreferences.
- 📝 **Notes** – Add and store custom travel notes.
- 📖 **Trip History** – Review all your past adventures in one place.

---

## 🛠️ Tech Stack

- **Kotlin (Android)**
- **Firebase Firestore & Authentication**
- **Google Maps SDK & Directions API**
- **Open-Meteo API** (Weather)
- **ExchangeRate-API** (Currency)
- **SharedPreferences** (Local storage for notes and packing list)
